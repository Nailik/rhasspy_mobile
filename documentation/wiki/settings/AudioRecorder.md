Can be set by calling [[Webserver]] ([[HttpApi]]) or [[MqttApi]].

| Setting    | Information                                      | 
|------------|--------------------------------------------------|
| Channel    | how many channels audio is recorded on           | 
| Encoding   | Encoding (PCM 16 Bit for Porcupine [[WakeWord]]) | 
| SampleRate | SampleRate for recording audio                   |
